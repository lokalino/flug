# Airline Booking System

This is a simple online airline ticket booking system built with React.

## Features

- Search for available flights
- Book airline tickets
- Create and login to user accounts
- Manage user account
- Add funds to user account

## Technologies

- **React**: Frontend library for building user interfaces
- **JSON Server**: Simple backend server for simulating a REST API
- **React Router**: Library for routing in React applications
- **CSS**: Styling the application

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/your-username/my-airline-booking-app.git
    cd my-airline-booking-app
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Start the JSON server:

    ```bash
    npx json-server --watch db.json --port 3001
    ```

4. Start the application:

    ```bash
    npm start
    ```

The application will be available at `http://localhost:3000`.

## Usage

1. Register a new user account.
2. Log in with your credentials.
3. Search for available flights.
4. Book a flight by selecting available options.
5. Add funds to your account if necessary.
6. Manage your bookings and account details.

# Buchungssystem für Flugtickets

Dies ist ein einfaches Online-Buchungssystem für Flugtickets, erstellt mit React.

## Funktionen

- Suche nach verfügbaren Flügen
- Buchung von Flugtickets
- Erstellen und Anmelden von Benutzerkonten
- Verwaltung des Benutzerkontos
- Hinzufügen von Guthaben zum Benutzerkonto

## Technologien

- **React**: Frontend-Bibliothek zur Erstellung von Benutzeroberflächen
- **JSON Server**: Einfacher Backend-Server zur Simulation einer REST-API
- **React Router**: Bibliothek für das Routing in React-Anwendungen
- **CSS**: Styling der Anwendung

## Installation

1. Klonen Sie das Repository:

    ```bash
    git clone https://github.com/your-username/my-airline-booking-app.git
    cd my-airline-booking-app
    ```

2. Installieren Sie die Abhängigkeiten:

    ```bash
    npm install
    ```

3. Starten Sie den JSON-Server:

    ```bash
    npx json-server --watch db.json --port 3001
    ```

4. Starten Sie die Anwendung:

    ```bash
    npm start
    ```

Die Anwendung ist unter `http://localhost:3000` verfügbar.

## Verwendung

1. Registrieren Sie ein neues Benutzerkonto.
2. Melden Sie sich mit Ihren Anmeldedaten an.
3. Suchen Sie nach verfügbaren Flügen.
4. Buchen Sie einen Flug, indem Sie verfügbare Optionen auswählen.
5. Fügen Sie bei Bedarf Guthaben zu Ihrem Konto hinzu.
6. Verwalten Sie Ihre Buchungen und Kontodaten.
